# Confluence A4 PDF Export App

Confluence 페이지를 A4 용지 크기에 최적화하여 PDF로 내보내는 애플리케이션입니다.

## 주요 기능

1. Confluence 페이지를 A4 용지 크기에 최적화하여 PDF로 내보내기
2. 실시간 A4 미리보기 기능
3. 여백, 타이틀 크기, 폰트, 헤더/푸터 등 PDF 출력 설정 조정 기능

## 설치 방법

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 빌드
npm run build
```
